# ragrev

Experiments reverse engineering Ragnarok and playing with rAthena.

* Patched 2006 Ragnarok Client from <https://archive.org/details/ragnaroklevelup2006> to disable GameGuard. Patched it's correspondent eAthena server to handle the Brazilian packet handshake.
* Developed a very simple network proxy `proxy.py` to analyze traffic between the client and the server.
